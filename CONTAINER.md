# MQTT Exporter Container Images

Container images based on [mqtt-exporter](https://github.com/kpetrem/mqtt-exporter), a generic Prometheus exporter that subscribes to MQTT topics and exposes them as metrics.

Sources are available on [GitHub](https://github.com/anthochamp/container-mqtt-exporter).

See [README.md](README.md) for full documentation and configuration reference.

## Image tags

- `x.y.z`: Container image version `x.y.z`.
- `edge`: Latest commit build.

**Tag aliases:**

- `x.y`: Latest patch of `x.y`.
- `x`: Latest minor+patch of `x`.
- `latest`: Latest `x.y.z` tag.
